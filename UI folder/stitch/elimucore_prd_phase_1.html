# ElimuCore Product Requirements Document (Phase 1) ## Overview ElimuCore is an offline-first learning platform for rural Uganda, featuring a local AI tutor (ElimuBot via Ollama). ## Design Language: Neumorphism - **Backgrounds**: #E0E5EC (or similar soft grey/blue). - **Elevated Surfaces**: Soft outer shadows (light: white, dark: soft grey-blue). - **Sunken Surfaces**: Inset shadows for inputs and pressed buttons. - **Rounding**: 16px to 24px for cards; 12px for buttons. ## Phase 1 Screens ### 1. Welcome / App Landing - **Goal**: Establish the app as offline-private and start the journey. - **Key UI**: Hero panel with ElimuCore branding, "Work Offline" indicator, Language toggle (EN/KS). ### 2. Role Selection - **Goal**: Segment users into Student, Facilitator, or Admin flows. - **Key UI**: Three large neumorphic cards with descriptive icons. ### 3. Login / Local Profile Access - **Goal**: Access local data without internet. - **Key UI**: Recent profile avatars (for students) and a secure form for staff. ### 4. Student Dashboard - **Goal**: Provide a clear "What's Next" for the learner. - **Key UI**: Progress cards, ElimuBot quick-access, and course highlights. ### 5. Course Catalog - **Goal**: Explore available skills. - **Key UI**: Search bar (inset), category filters, and course cards with progress indicators.
