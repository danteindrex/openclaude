Understanding Epsilon (ε) and Delta (δ)

Think of it as a challenge-response game. Epsilon (ε) is how close you want the output to be to the limit, and Delta (δ) is how close the input needs to be to guarantee that.

// The formal definition
∀ε > 0, ∃δ > 0 such that
0 < |x - c| < δ ⇒ |f(x) - L| < ε
