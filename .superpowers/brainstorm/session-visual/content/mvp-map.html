Recommended MVP

  • Pixel-faithful Next.js shell from stitched UI
  • Real Ollama tutor chat as the product centerpiece
  • Dashboard / catalog / lessons rendered with demo data
  • Terminal capabilities surfaced as inspectable UI panels

Not Recommended For MVP

  • Rebuilding every CLI command as a polished web workflow at once
  • Full auth, multi-role backend, and production data model together
  • Perfect parity with all CLI-only internals in one pass
